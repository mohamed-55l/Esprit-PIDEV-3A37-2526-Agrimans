Objectifs 
Améliorer les performances de votre application (intégrité, sécurité et configuration) 
 
Travail demandé : 
1- Installer le bundle doctor-doctrine à travers cette commande : 
▪ 
composer require --dev ahmed-bhs/doctrine-doctor 
▪ 
Voir le lien github suivant pour plus de détails : 
https://github.com/ahmed-bhs/doctrine-doctor 
 
Au cas où vous rencontrez l’erreur suivante : 
 
Figure 1 : Problème d’installation qui peut se produire 
▪ 
Taper la commande : 
composer require ahmed-bhs/doctrine-doctor:^1.0 webmozart/assert:^1.11 --
with-all-dependencies 
▪ 
Ensuite taper la commande 
composer require --dev ahmed-bhs/doctrine-doctor 
2- Accéder à n’importe quelle page dans votre application symfony 
(dev environment) 
3- Ouvrir Symfony Web Profiler (toolbar en bas) 
4- Cliquer sur "Doctrine Doctor" panel
