US-003 S2 USER
En tant qu'utilisateur, je veux laisser un avis sur un équipement pour partager mon expérience
Étant donné
l'utilisateur 'Mohamed' ayant utilisé l'équipement 'Tracteur John Deere' disponible dans la plateforme
Quand
il soumet le formulaire avec le commentaire "Excellent équipement" et une note de 5 étoiles
Alors
l'avis est enregistré avec date_review = now(), Mohamed est redirigé vers la liste des avis
Étant donné
l'utilisateur 'Aziz' sur le formulaire d'avis pour l'équipement 'Tracteur John Deere'
Quand
il soumet le formulaire sans renseigner la note (champ obligatoire vide)
Alors
le formulaire est réaffiché avec l'erreur "La note est requise", aucune donnée n'est sauvegardée en base
